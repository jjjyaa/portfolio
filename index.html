PROFILE

안녕하세요 사용자와 기술을 연결하는 웹 풀스택 신입 개발자 최희주 입니다.

스마트공장 프로젝트 실무에서 개발자들과 협업하며 개발의 몰입감과 성취감을 처음 느꼈습니다.
이후 직접 코드를 배우기 시작했고 국비개발과정 수료를 통해 웹 개발 전반(Spring, Vue, React 등)을 실습하며 실무 역량을 키웠습니다.
개발자로서 저는 단순히 기능을 만드는 것을 넘어서 사용자의 흐름과 문제를 해결하는 개발자가 되고자 합니다.
기획, 협업, 구현까지 전 과정을 경험한 만큼 기술과 사람 사이를 연결할 수 있는 개발자로 성장하고 싶습니다.
웹 게시판과 쇼핑몰 프로젝트를 통해 프론트와 백엔드를 아우르는 구조 설계와 개발을 주도했고 AI/제조 데이터를 다루는 현장 경험은 데이터 기반 시스템에도 자신감을 더해주었습니다.

지금 이 순간도 기술로 가치를 만드는 개발자가 되기 위해 꾸준히 도전 중입니다.
저는 문제 앞에서 피하지 않고 사람과 기술을 연결하는 팀에 꼭 필요한 개발자입니다.
